# Documentation Technique - Projet JavaScript Exemple

## Présentation
Ce projet est une application JavaScript simple permettant d'afficher "Hello World" dans la console.

## Structure du projet
- `src/` : Contient le code source
  - `index.js` : Point d'entrée de l'application

## Installation
```bash
npm install
```

## Lancement
```bash
npm start
```

## Règles de développement
- Respecter les principes Clean Code (DRY, KISS, SOLID)
- Commenter le code si nécessaire
- Utiliser des noms de variables explicites

## Exemple de code
```javascript
// src/index.js
console.log('Hello World');
```
